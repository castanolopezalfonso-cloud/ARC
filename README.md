# ARC - Aspect Ratio Calculator

Esta carpeta contiene todo lo necesario para subir la calculadora de Fluge a GitHub y hostearla gratuitamente como una aplicación online en **Streamlit Cloud**.

## Pasos para desplegar en Streamlit 🚀

1. **Sube esta carpeta a GitHub:**
   Crea un nuevo repositorio en tu cuenta de GitHub y sube los 3 archivos de esta carpeta (`app.py`, `index.html`, `requirements.txt`).

2. **Conecta con Streamlit:**
   - Ve a [share.streamlit.io](https://share.streamlit.io/) e inicia sesión vinculando tu cuenta de GitHub.
   - Haz clic en **New app**.
   - En **Repository**, selecciona el repositorio que acabas de crear.
   - En **Branch**, asegúrate de que sea `main` (o `master`).
   - En **Main file path**, escribe `app.py`.

3. **Desplegar:**
   Haz clic en **Deploy!** y en un par de minutos tu calculadora estará viva online y accesible para todo tu equipo.

---
*💡 Nota Pro: Ya que toda la magia y el código espectacular que hemos hecho funciona directamente en JS/HTML en el navegador, si quieres una web aún más rápida, puedes activar "GitHub Pages" en la configuración de tu repositorio, apuntando directamente a `index.html`. ¡Funcionará como nativa sin requerir servidor Python!*
